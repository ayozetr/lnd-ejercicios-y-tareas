# Ejercicio 2

__Los estándares HTML 5.1 y HTML 5.2 introducen etiquetas HTML nuevas. Busca dos o tres etiquetas HTML de cada estándar HTML 5.1 y 5.2 y comprueba si tu versión de navegador Google Chrome las soporta. Para comprobar si tu versión del navegador soporta las etiquetas HTML usa la siguiente URL:__

HTML 5.1:

    <main>: Se introdujo para definir el contenido principal de un documento
    
    <nav>: Se añadió para definir un bloque principal o conjunto de enlaces de navegación
    
    <article>: Se incorporó para contenido independiente que puede colocarse de forma autónoma en el sitio web
    

HTML 5.2:

    <dialog>: Esta es la característica principal de HTML 5.2. Permite crear cuadros de diálogo o modales de forma nativa1
    
    <menu> y <menuitem>: Aunque estas etiquetas se introdujeron inicialmente, fueron eliminadas en HTML 5.2
    
    <legend>: Si bien esta etiqueta ya existía, en HTML 5.2 se modificó para permitir elementos hijos, como encabezados (por ejemplo, <h1>, <h2>, etc.) dentro de ella
